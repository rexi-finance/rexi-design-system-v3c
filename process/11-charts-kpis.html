09 — Implementation Notes
How the glass design system maps to the existing Recharts + shadcn/ui chart architecture.
Current Change Impact
--chart-1 to --chart-6 Replace steel grays with semantic palette in globals.css All charts auto-update. Zero component changes.
ChartContainer CSS selectors Update grid/tick/axis colors to glass tokens chart.tsx — 3-4 class changes
ChartTooltipContent Style with bg-elevated, glass border, deep shadow chart.tsx — tooltip classes
AreaChart gradients Already use stop-opacity 80%→10% — adjust to 30%→2% AreaChart.tsx — 2 values
KPICard Replace bg-card/border-border with glass tokens KPICard.tsx — Tailwind classes
CHART_COLORS array Update to reference new --chart-* variables chart-config.ts — already uses vars
Bottom line: The existing architecture is fully compatible. The changes are purely CSS variable swaps and a few Tailwind class updates. No structural changes to components, no API changes, no data flow changes.
