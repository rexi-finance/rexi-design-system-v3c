Focus State — 3 propuestas

El focus state es lo que más impacta cómo se siente interactuar con un input. Las 3 propuestas usan la misma base pero difieren en cómo marcan el foco.

A — BORDER ONLY — glass coherent

El border sube de 6% a 14% al focus. Misma lógica que los botones: más border = más activo. Sin color de marca.

Stripe settlem

border: 6% → 14% (glass-border-hover)
fill: sin cambio · no ring · no color

Pro: 100% glass coherent
Con: Sutil — puede ser difícil de notar en algunos monitores. Accessibility score bajo sin focus ring visible.
Accessibility concern
B — BLUE BORDER — brand focus

El border cambia a accent-blue en focus. Convención estándar (Figma, Linear, Vercel). El color de marca marca el foco.

Pro: Claro, familiar, buena accessibility
Con: Introduce color al sistema glass. ¿Viola P4 (color = información)? Blue = brand, pero ¿focus es info?
Accessibility OK Color = info tension
C — BORDER + RING — glass + a11y

Border sube a 14% (glass coherent) + un focus ring sutil de blue. La border mantiene glass; el ring da accessibility.

Regla P4: Blue = brand/neutral action. Los controles activos usan blue porque representan una acción/selección del usuario, no información del sistema.

Resumen — Focus State:

A — Border Only: Máxima coherencia glass. Pero la diferencia entre resting (6%) y focus (14%) puede ser sutil en monitores de baja calidad. Accessibility score bajo — WCAG recomienda un focus indicator visible y distinguible.

B — Blue Border: Más visible y familiar (Figma, Linear, Vercel usan blue focus). Pero introduce color en un lugar que no es estrictamente "información" — es un estado de interacción. En un UI con pocos colores, puede ser un acento agradable o una distracción.

C — Border + Ring: Best of both worlds. La border se mantiene glass (14%). El ring blue es tan sutil (20% opacity) que no distrae pero cumple accessibility. Es lo que usa shadcn/ui de base. Dos señales puede parecer redundante, pero cada una tiene un propósito distinto: border = glass system, ring = accessibility.

Mi recomendación: C — el glass system se mantiene intacto en los borders, y el focus ring sutil da accessibility sin romper la estética. Es el approach más robusto.
