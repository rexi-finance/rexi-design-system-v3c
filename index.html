6 Design Principles
P1
Narrate, don't list
Every number answers a question. Match Rate is the answer to "how is my reconciliation going?"
P2
Space is the design
48px content padding. 40px between sections. The interface breathes so the user can think.
P3
Subtract, don't add
No card wrappers on focal metrics. Every border, shadow, and container must justify its existence.
P4
Color is information
Never decorative. Emerald = healthy. Amber = attention. Red = error. Blue = brand. Purple = integration.
P5
One focal point
Each screen has ONE dominant element. Everything else is visually subordinate.
P6
Typography carries personality
Figtree 900 for hero numbers. Merriweather 300 italic for accent moments.
